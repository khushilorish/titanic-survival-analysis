**Titanic Survival Analysis**

Objective

The goal of this project is to analyze the Titanic dataset and identify the key factors that influenced passenger survival.

Workflow

- Data Cleaning (handling missing values, dropping irrelevant columns)
- Feature Engineering (FamilySize, FamilyType)
- Exploratory Data Analysis (EDA)
- Feature Interaction Analysis (Sex, Pclass, Family)

Key Insights

- Gender is the most influential factor — females had significantly higher survival rates.
- First-class passengers had better survival chances than lower classes.
- Small families (1–3 members) had the highest survival rates.
- Males in lower classes and large families had the lowest survival probability.

Tools Used

- Python
- Pandas
- NumPy
- Seaborn / Matplotlib
