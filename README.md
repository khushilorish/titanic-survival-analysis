**Titanic Survival Analysis**

**Overview**

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify the key factors influencing passenger survival.

**Key Findings**

- Gender is the most significant factor — females had much higher survival rates than males.
- Passenger class strongly influenced survival, with first-class passengers having the highest chances.
- Small families (1–3 members) had better survival rates than solo travelers or large families.
- Males in lower classes and large families had the lowest survival probability.

**Techniques Used**

- Data Cleaning (handling missing values, dropping irrelevant features)
- Feature Engineering (FamilySize, FamilyType)
- Exploratory Data Analysis (groupby, visualizations)
- Feature Interaction Analysis (Sex, Pclass, Family)

**Tools**

- Python
- Pandas
- NumPy
- Seaborn / Matplotlib

**Project Structure**

- "notebook/" → Jupyter notebook with full analysis
- "data/" → dataset used
